# 雪球适配器

## 目标

雪球是果汁哥投资内容的核心阵地，发布股市分析、宏观观点和投资策略。内容价值密度高，是主题查询（美股/A股/黄金/比特币等）的优先查询平台。

## 官方账号

- **账号名**：juicebro
- **平台**：雪球（xueqiu.com）
- **主页 URL**：`https://xueqiu.com/u/juicebro`
- **搜索入口**：`https://xueqiu.com/search?q=juicebro&type=user`
- **内容类别**：投资主题

## 查询策略

### 主要查询路径

1. **用户时间线**：访问 `xueqiu.com/u/juicebro`，获取动态列表
2. **雪球 API**（可选，V2 扩展点）：
   - 用户动态：`https://xueqiu.com/v4/statuses/user_timeline.json?user_id={uid}&count=20`（需 Cookie）
3. **Web 直接访问**（推荐）：雪球用户主页公开可访问，无需登录

### 内容类型识别

| 雪球内容 | content_type |
|----------|-------------|
| 雪球动态（短评论） | `short_post` |
| 雪球文章（长文） | `article` |
| 转发（带评论） | `short_post`（标注为转发） |

## 适合提取的字段

| 字段 | 可用性 | 提取说明 |
|------|--------|---------|
| `publish_time` | ✅ 高 | 雪球显示绝对时间，格式如"2024-01-15 10:30" |
| `title` | ✅ 中 | 文章有标题；短动态无标题（设为 null） |
| `summary` | ✅ 高 | 动态正文前 150 字 |
| `content_type` | ✅ 高 | 文章 / 动态 / 转发 类型明确 |
| `source_locator` | ✅ 高 | 格式：`https://xueqiu.com/{uid}/{status_id}` |
| `like_count` | ✅ 高 | 点赞数 |
| `comment_count` | ✅ 高 | 评论数 |
| `repost_count` | ✅ 高 | 转发数 |

## "今天"查询处理

1. 访问用户时间线，动态按发布时间降序排列
2. 雪球时间格式为绝对时间，直接与当日日期比对
3. 若时间线第一条非当日，说明今日无更新
4. 雪球每日可能发多条动态，全部收录

## 跨平台去重

- **与今日头条/公众号**：长文分析可能同步，以标题去重
- **与微博**：短评论可能同步，以前100字相似度判断
- 雪球内容投资属性强，通常为独家内容，重复率较低

## 失败处理

| 失败场景 | 处理方式 |
|----------|----------|
| 用户主页访问正常，API 失败 | 降级为 Web 端内容提取 |
| 主页返回空列表 | 说明"雪球账号 juicebro 近期无动态更新" |
| 内容被折叠（需点击展开）| 仅提取可见部分，标注"内容已折叠，可能不完整" |
| 账号不存在 | 说明 URL 可能有误，建议用户手动搜索 juicebro |

## 注意事项

- 雪球的账号 ID 和用户名可能不同，URL 中用的是用户名（juicebro）
- 雪球的 API 通常需要 Cookie 认证，直接 Web 访问更稳妥
- 转发内容（带原文）应提取原文摘要而非转发评论
- 投资内容具有时效性，查询时注意标注内容发布时间
