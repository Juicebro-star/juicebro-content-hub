# 元宝适配器

## 目标

元宝（腾讯元宝）是果汁哥的新兴内容平台，目前处于探索阶段。本适配器采取保守策略，明确标注访问能力的不确定性。

## 官方账号

- **账号名**：果汁哥
- **平台**：元宝（腾讯元宝）
- **访问方式**：主要通过元宝 App
- **内容类别**：日常分享

## 查询策略

### 当前状态

元宝是腾讯推出的 AI 内容平台，作为 C 端内容分发产品目前仍在快速迭代中。Web 端访问能力尚不明确，公开 API 暂不可用。

### 主要查询路径

1. **元宝 App**：最直接的访问方式，在 App 内搜索"果汁哥"
2. **Web 端**（能力待确认）：尝试访问元宝 Web 端用户主页
3. **腾讯内容聚合**：元宝内容可能通过微信生态分发，部分内容可能在微信或公众号中找到

## 适合提取的字段

| 字段 | 可用性 | 提取说明 |

| `publish_time` | ⚠️ 待确认 | 取决于 App/Web 端的展示 |

## "今天"查询处理

由于元宝的 Web 访问能力尚不明确：
1. 尝试访问元宝 Web 端，若成功则按标准流程处理
2. 若 Web 端访问受限，标注受限状态
3. 不强制查询，在全平台汇总中标注元宝为"访问能力待确认"

## 跨平台去重

- 元宝内容可能与微信公众号或其他腾讯生态内容重叠
- 去重策略与其他平台相同：以标题+摘要前100字为去重键

## 失败处理

| 失败场景 | 处理方式 |
|----------|----------|
| Web 端不可访问 | 输出：元宝（腾讯元宝）平台目前 Web 端访问受限，建议在元宝 App 内搜索"果汁哥" |
| 账号找不到 | 说明该账号信息来源于账号矩阵，实际账号可能有变动 |

## 注意事项

- 本适配器采取"保守探测"策略：先尝试访问，若失败则透明说明
- 元宝作为新兴平台，账号矩阵数据（账号名、URL）可能随平台迭代而变化
- 建议定期确认元宝账号的实际访问方式，并更新本适配器
