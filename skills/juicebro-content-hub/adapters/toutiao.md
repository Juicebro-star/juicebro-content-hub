# 今日头条适配器

## 目标

今日头条是果汁哥发布深度文章的平台之一，内容以财经分析和知识分享为主，适合提取长文内容。

## 官方账号

- **账号名**：果汁哥的知识分享
- **平台**：今日头条（toutiao.com）
- **主页 URL**：`https://www.toutiao.com/c/user/token/{token}/`（需先确认 token）
- **搜索入口**：`https://so.toutiao.com/search?keyword=果汁哥的知识分享&pd=user`
- **内容类别**：投资主题

## 查询策略

### 主要查询路径

1. **作者主页文章列表**：访问今日头条作者主页，获取文章列表
2. **头条搜索**：通过搜索确认账号，筛选"果汁哥的知识分享"的内容
3. **微头条**：短内容（微头条）和文章分开展示，都需查询

### 内容类型识别

| 头条内容 | content_type |
|----------|-------------|
| 图文文章（> 500 字） | `article` |
| 微头条（短内容） | `short_post` |
| 视频 | `video` |

## 适合提取的字段

| 字段 | 可用性 | 提取说明 |
|------|--------|---------|
| `publish_time` | ✅ 高 | 文章列表中显示发布时间 |
| `title` | ✅ 高 | 文章标题清晰 |
| `summary` | ✅ 中 | 文章摘要或前段 |
| `content_type` | ✅ 高 | 文章 / 微头条 / 视频 类型明确 |
| `source_locator` | ✅ 高 | 格式：`https://www.toutiao.com/article/{article_id}` |
| `view_count` | ✅ 高 | 阅读量 |
| `like_count` | ✅ 中 | 点赞数 |

## "今天"查询处理

1. 访问作者主页文章列表（按时间降序）
2. 筛选当日发布的内容
3. 头条时间格式：
   - "X分钟前" / "X小时前"：换算
   - "MM-DD"：当年日期
   - "YYYY-MM-DD"：历史内容

## 跨平台去重

- **与微信公众号**：文章内容通常同步，以标题去重，保留头条版本（阅读量数据更公开）
- **与雪球**：长文分析可能同步，以标题去重

## 失败处理

| 失败场景 | 处理方式 |
|----------|----------|
| 主页需要登录 | 说明：头条部分内容需登录查看，建议在头条 App 搜索"果汁哥的知识分享" |
| 文章详情无法访问 | 仅提取列表中的标题和摘要 |
| 搜索结果有多个同名账号 | 说明存在多个同名账号，无法确认正确账号时提示用户手动确认 |

## 注意事项

- 今日头条账号 URL 中的 token 是数字串，需通过搜索确认
- 同名账号"果汁哥的知识分享"在头条和抖音均使用，确认时注意区分
- 头条文章可能有平台专属内容，也可能同步自其他平台
