# 抖音适配器

## 目标

抖音是果汁哥的核心视频平台之一，以知识分享类短视频为主，账号名"果汁哥的知识分享"。

## 官方账号

- **主账号名**：果汁哥的知识分享
- **备用账号名**：Roaring_Kitty
- **平台**：抖音（douyin.com）
- **主页 URL**：`https://www.douyin.com/user/{uid}`（UID 需先确认）
- **搜索入口**：在抖音 Web 或 App 搜索"果汁哥的知识分享"
- **内容类别**：日常分享

## 查询策略

### 主要查询路径

1. **用户主页视频列表**：访问 `douyin.com/user/{uid}`，获取视频列表
2. **抖音搜索**：`https://www.douyin.com/search/果汁哥的知识分享?type=user`
3. **两个账号都需查询**：主账号（果汁哥的知识分享）+ 备用账号（Roaring_Kitty），合并结果后去重

### 内容类型识别

| 特征 | content_type |
|------|-------------|
| 竖屏短视频（< 3 分钟） | `video` |
| 较长视频（> 3 分钟） | `video` |
| 图文合集（多图滑动） | `image_post` |

## 适合提取的字段

| 字段 | 可用性 | 提取说明 |
|------|--------|---------|
| `publish_time` | ✅ 高 | 视频发布日期，Web 端可见 |
| `title` | ✅ 高 | 视频标题（通常为视频描述前几十字） |
| `summary` | ✅ 中 | 视频描述文字 |
| `content_type` | ✅ 高 | 默认为 video |
| `source_locator` | ✅ 高 | 格式：`https://www.douyin.com/video/{video_id}` |
| `view_count` | ✅ 高 | 播放量（公开显示） |
| `like_count` | ✅ 高 | 点赞数 |
| `duration` | ✅ 中 | 视频时长 |
| `thumbnail_url` | ✅ 高 | 封面图 URL |

## "今天"查询处理

1. 访问用户主页，视频列表默认时间降序
2. 检查前 5-10 条视频的发布时间
3. 抖音的时间格式：
   - "MM-DD"：当年该日期（无时间精度）
   - "YYYY-MM-DD"：历史内容
   - "X天前"：近期相对时间
4. 若最近一条视频发布日期非今日，输出"今日无新视频"

## 跨平台去重

- **与快手**：内容高度重叠，通常同步发布，以视频标题为去重键
- **与 bilibili**：部分视频同步，标题可能稍有差异，以前 20 字相似度判断
- 去重策略：抖音 + 快手视为同源，保留抖音版本，标注 `also_posted_on: ["kuaishou"]`
- 若 bilibili 有更长版本，保留 bilibili 版本，抖音版本标注 `also_posted_on: ["bilibili"]`

## 失败处理

| 失败场景 | 处理方式 |
|----------|----------|
| Web 端访问被重定向至 App | 说明：抖音 Web 端部分内容需通过 App 查看。请在抖音 App 搜索"果汁哥的知识分享" |
| 视频列表无法加载 | 记录用户主页基本信息（若有），标注"视频列表无法加载" |
| UID 未确认 | 通过搜索入口重新定位，不得使用猜测 UID |
| 两个账号均无法访问 | 说明两个账号名并建议用户自行搜索 |

## 注意事项

- Roaring_Kitty 是备用账号名，需同时查询以确保内容覆盖完整
- 两个账号可能发布不同内容，不可假设内容完全相同
- 抖音视频的 `video_id` 通常出现在分享链接中
