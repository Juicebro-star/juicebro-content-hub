# 喜马拉雅适配器

## 目标

喜马拉雅是果汁哥播客内容的主要分发平台之一，音频内容为主，受众广泛。内容与小宇宙高度重叠，两个平台通常同步更新。

## 官方账号

- **账号名**：果汁哥juicebro
- **平台**：喜马拉雅（ximalaya.com）
- **主播主页**：`https://www.ximalaya.com/zhubo/{uid}`（UID 需确认）
- **搜索入口**：`https://www.ximalaya.com/search?q=果汁哥juicebro&type=0`
- **内容类别**：播客

## 查询策略

### 主要查询路径

1. **主播主页专辑列表**：访问主播主页，查看专辑和节目列表
2. **专辑内节目列表**：进入具体专辑，获取节目列表（按更新时间排序）
3. **喜马拉雅搜索**：搜索主播名确认账号，避免同名主播混淆
4. **RSS 订阅**（V2 扩展点）：喜马拉雅部分专辑支持 RSS，可通过 RSSHub 获取

### 内容类型识别

| 喜马拉雅内容 | content_type |
|-------------|-------------|
| 播客节目（音频） | `audio` |
| 声音（短音频） | `audio` |

## 适合提取的字段

| 字段 | 可用性 | 提取说明 |
|------|--------|---------|
| `publish_time` | ✅ 高 | 节目列表中显示更新时间 |
| `title` | ✅ 高 | 节目标题 |
| `summary` | ✅ 高 | 节目简介 |
| `content_type` | ✅ 高 | 固定为 audio |
| `source_locator` | ✅ 高 | 格式：`https://www.ximalaya.com/sound/{sound_id}` |
| `play_count` | ✅ 高 | 播放次数 |
| `duration` | ✅ 高 | 音频时长（格式：MM:SS 或 HH:MM:SS） |
| `episode_number` | ✅ 中 | 专辑内节目序号（如：第 42 期） |

## "今天"查询处理

1. 访问专辑节目列表，默认时间降序
2. 检查最近节目的发布时间
3. 喜马拉雅时间格式：
   - "MM-DD"：当年发布
   - "YYYY-MM-DD"：历史内容
   - "X天前"：相对时间
4. 播客通常为周更或不定期更新，今日无更新属正常情况

## 跨平台去重

- **与小宇宙**：内容高度重叠（通常同步发布）
- 去重依据：节目标题 + 时长相似（±10秒）
- 去重策略：优先保留小宇宙版本（因为小宇宙是主播自平台），喜马拉雅标注 `also_posted_on: ["xiaoyuzhou"]`
- 若仅喜马拉雅有某节目，保留喜马拉雅版本

## 失败处理

| 失败场景 | 处理方式 |
|----------|----------|
| 主播主页无法访问 | 说明：喜马拉雅访问受限，建议在喜马拉雅 App 搜索"果汁哥juicebro" |
| 专辑无法找到 | 通过搜索重新定位主播 |
| 付费音频 | 仅提取标题和简介，标注"[付费内容]"，不尝试获取音频内容 |
| 与小宇宙内容完全重叠 | 合并处理，在小宇宙条目中标注 also_posted_on: ximalaya |

## 注意事项

- 喜马拉雅的播客可能分布在多个专辑下，需要全部查询
- 部分节目为付费内容，仅能获取标题和时长，无法访问音频
- 喜马拉雅的主播 UID 需要从用户主页 URL 中提取
