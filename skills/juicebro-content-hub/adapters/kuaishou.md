# 快手适配器

## 目标

快手是果汁哥的视频分发平台之一，内容与抖音高度重叠。主要作为备用查询平台，当抖音访问受限时提供替代数据源。

## 官方账号

- **账号名**：果汁哥
- **平台**：快手（kuaishou.com）
- **主页 URL**：`https://www.kuaishou.com/profile/{uid}`（UID 需先确认）
- **搜索入口**：在快手 Web 或 App 搜索"果汁哥"
- **内容类别**：日常分享

## 查询策略

### 主要查询路径

1. **用户主页视频列表**：访问快手用户主页，获取视频列表
2. **快手搜索**：`https://www.kuaishou.com/search/video?searchKey=果汁哥` 结合用户过滤
3. **快手 App**：App 内搜索体验优于 Web 端

### 内容类型识别

| 特征 | content_type |
|------|-------------|
| 短视频 | `video` |
| 图文合集 | `image_post` |

## 适合提取的字段

| 字段 | 可用性 | 提取说明 |
|------|--------|---------|
| `publish_time` | ✅ 中 | 快手显示相对时间，需换算 |
| `title` | ✅ 高 | 视频描述（通常与抖音标题相同） |
| `summary` | ✅ 中 | 视频描述文字 |
| `content_type` | ✅ 高 | 默认为 video |
| `source_locator` | ✅ 高 | 格式：`https://www.kuaishou.com/short-video/{video_id}` |
| `view_count` | ✅ 高 | 播放量 |
| `like_count` | ✅ 高 | 点赞数 |
| `duration` | ✅ 中 | 视频时长 |

## "今天"查询处理

1. 快手视频列表按时间降序排列
2. 时间显示格式：
   - "X分钟前" / "X小时前"：换算为绝对时间
   - "昨天"：前一天（具体时间待确认）
   - "X天前"：相对天数
3. 若最新视频非当日发布，输出"今日暂无新视频"

## 跨平台去重

- **与抖音**：同一条视频几乎必然同时发布，视为同源
- 去重处理：通常优先保留抖音版本，快手填入 `also_posted_on`
- 若仅快手有该视频（抖音未发布），则保留快手版本

## 失败处理

| 失败场景 | 处理方式 |
|----------|----------|
| Web 端访问受限 | 说明：快手内容建议在快手 App 内查看，搜索用户"果汁哥" |
| 与抖音内容完全重叠 | 直接标注"快手内容已合并至抖音版本，请参见上方抖音区块" |
| 账号搜索结果有多个 | 注意：快手"果汁哥"账号名可能有多个创作者使用，确认时检查头像/简介 |

## 注意事项

- 快手和抖音内容重叠率极高，全平台查询时可考虑将两者视为同一内容源处理
- 快手账号名"果汁哥"较为通用，确认账号时需额外验证（头像、粉丝数、简介）
- 快手 Web 端功能相对有限，App 体验更佳
